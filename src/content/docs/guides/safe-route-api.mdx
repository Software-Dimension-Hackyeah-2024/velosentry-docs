---
title: Safe Route API
description: A guide to using the VeloSentry Safe Route API for finding safe bike paths with safety score and route details.
---

The _Route API_ determines few possible bike routes through points 0 to N-1.

```sh
curl 'https://velosentry.io/safe-route/[lon0],[lat0];[lon1],[lat1];...;[lonN-1],[latN-1]'
```

## Parameters:

The endpoint requires all the points the route should go through in a format [longitude],[latitude]

## Response:

In addition to the `/route` endpoint, the endpoint return safety score based on the road type, dangerous intersections and maximum speed. Additionally, it returns all available data about road quality (surface, condition, lights).
