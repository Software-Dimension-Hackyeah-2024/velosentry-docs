---
title: Route API
description: A guide to using the VeloSentry Route API for finding safe bike paths.
---

The _Route API_ determines few possible bike routes through points 0 to N-1.

```sh
curl 'https://velosentry.io/route/[lon0],[lat0];[lon1],[lat1];...;[lonN-1],[latN-1]'
```

## Parameters:

The endpoint requires all the points the route should go through in a format [longitude],[latitude]

## Response:

For more information on the response details please refer to [OSRM documentation](https://project-osrm.org/docs/v5.24.0/api/#match-service).
