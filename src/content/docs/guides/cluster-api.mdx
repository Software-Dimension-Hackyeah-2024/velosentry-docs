---
title: Cluster API
description: A guide to using the VeloSentry Cluster API.
---

The _Cluster API_ groups the accidents on the route and returns one coordinate for each cluster.

```sh
curl 'https://velosentry.io//cluster-accidents?coords=20.023949,50.084293;19.907637,50.027573&maxDistance=200&clusterDistance=200'
```

## Parameters

- `coords` - A list of coordinates that form the route
- `maxDistance` - maximal distance of the accident from the chosen route in meters
- `clusterDistance` - maximal radius in which the accidents are in the same cluster
